## 🔍 Overview

- **배경**: 유통·커머스에서 고객 구매 이력은 세그먼트 기반 개인화 마케팅의 출발점. 페르소나를 정의하고 행동을 확률 모형으로 설명하는 것이 핵심.
- **문제**: 구매 로그로부터 **고객 세그먼트를 정의**하고, **행동 확률 모델**로 개인화 전략을 제안.

## 🧠 Approach — 3-step Analytics

### 1️⃣ Persona Mapping
- 고객 행동 지표(RFM·카테고리·채널) 조합으로 페르소나 정의
- 대표 세그먼트 프로파일링

### 2️⃣ Bayesian Modeling
- 베이지안 추론으로 구매 확률·재구매 확률 모델링
- 불확실성 정량화 → 마케팅 우선순위 스코어링

### 3️⃣ EDA for User Segment
- 세그먼트별 특성 비교
- 개인화 마케팅 전략 수립
